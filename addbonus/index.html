Add Bonus

Add Bonus is a functionality which is used when the customer has already made their Internet package payment but due to some system issues, the NETTV records didn't get updated resulting in NETTV Service disruption. In this case, customer don't need to pay for NETTV separately because the NETTV is included with the Internet Package itself. NEED TO REVIEW and MAKE AMMENDMENTS

Note

Bonus is provided as a free package with certain WorldLink Internet packages. Amount is billed to the WorldLink and not the end customer as WorldLink acts as a reseller of NETTV.

How to Add Bonus?

To add bonus for eligible customers:

  1. Enter the Customer's username in the USERNAME field and select the respective username from the suggestions.
  2. Choose a respective MAC Address from Select MAC dropdown field.
  3. Related information such as Offer name, Stb and Offer duration will be loaded after selecting the desired MAC Address.
  4. Enter a valid internet invoice number in the Invoice field.
  5. Click the ADD button.

Criteria for the Bonus Addition

  • Number of Set top box (STB) defined in plan category are eligible. (Primary STB has higher priority)
  • Plan category ID should be mapped to NETTV offer.
  • Should have a valid internet invoice number.
  • Rental and refundable should be paid or adjusted.
  • STB should be associated with username on ebill as well as on NETTV end. In case of mismatch, bonus cannot be applied.
  • Nettv package is synced with the number of days of the internet days.
