Buy Package

Buy Package functionality allows users to purchase a NETTV package for a customer. This is useful for situations where the customer hasn't previously purchased a NETTV package or wishes to top-up their existing plan.

How to Buy/Top Up Package?

To buy package for customers:

  1. Enter the Customer's username in the USERNAME field and select the correct username from the suggestions.
  2. Choose the corresponding STB MAC Address from STB dropdown.
  3. Related information such as STB TYPE, MAPPING and PACKAGE will automatically populate upon selecting the MAC address.
  4. Select the desired NETTV package from the PACKAGE dropdown menu.
  5. After selecting a package, the TOP AMOUNT field will update with the price of the selected package.
  6. Click the ADD button.
  7. A confirmation prompt will appear. Click Continue to buy the package.
