Getting Started

Login to NETTV Dashboard

NETTV Dashboard can be accessed through a web browser via https://nettv.wlink.com.np. You will see a Login box asking for your WorldLink credentials.

Prerequisities

  • A valid WorldLink Exchange ID. (Exchange ID must have a valid email address.)
  • Exchange ID must exist in SMSCAST.
  • A valid NETTV Dashboard access. (Please contact your immediate supervisor if you need the access.)
