    Offers

    Offers section provides informations regarding the NETTV offers along with their details and requirements.

    Offers User Interface

    The user interface can be broken down into:

    1. Search: Currently the search is non-functional.
    2. NAME: Name of the NETTV Offer.
    3. DURATION: Duration of the respective offer.
    4. AMOUNT: Price of the Offer.
    5. STATUS: Shows whether the offer is Enabled or Disabled.
    6. CHARGE: Require means that the customer needs to pay the charged amount in order to get this offer, Skip means they do not need to pay.
    7. RENEWABLE: Shows whether the offer can be renewed or not after it's validity period ends.
    8. MULTIPLE INVOICE: ????????????????????

    NETTV Offer Map

    NETTV Offer Map section basically facilitates user to map NETTV offer to a single or multiple Internet Plan categories.

    How to map NETTV Offer?

    To map a NETTV offer with an Internet Plan category:

    1. In the Enter plan category id, enter the Internet plan category that needs to be mapped with an offer.
    2. In the Enter offer name, enter the name of the NETTV offer with which the above Internet category shall be mapped.
    3. Click MAP button to Map the Plan with the Offer.

    Note

    You can map select multiple Internet categories to map with a NETTV offer.

    Offers Mapping

    Offer Mapping sections shows the Offers mapped with the Internet categories along with other details.

    Offers Mapping User Interface

    The User Interface can be broken down into:

    1. Search: Search for Plan category through Plan category ID.
    2. PLAN CATEGORY ID: Shows the Id of the Internet Plan Category.
    3. PLAN NAME: Shows the name of the Plan Category.
    4. OFFER NAME: Shows the name of the NETTV Offer.
    5. OFFER DURATION: Shows the duration of the respective NETTV offer.
    6. SERVICES: Shows which service the offer provide.
    7. RENTAL CHARGE: Show the amount of Rental fee needed for the given offer.
    8. STATUS: Shows whether the offer is Enabled or Disabled.
