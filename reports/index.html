Reports

Reports

Reports is where all the records of performed/executed activities reside.

Action Log

Action Log sections stores the logs of the Users(WorldLink Staff) that performed certain actions on a certain Username on a certain date.

Action Log User Interface

The User Interface can be broken down into:

  1. Search: Search for Action through Customer Username.
  2. USERNAME: Displays the Username of the NETTV Customer.
  3. MAC ADDRESS: Displays the Device/STB MAC Address of the respective username.
  4. EXCHANGE USERNAME: Displays the exchange username of the WorldLink's staff who performed the action.
  5. ACTION DATE: Shows when the respective action was performed.
  6. ACTION: Shows the action that was executed. Actions can either be create or delete.

Bonus List

Bonus List section displays the logs of the bonus given to the NETTV customers when they bought a certain Internet package that included a NETTV package as a bonus.

Note

Bonus is added automatically during the purchase of Internet package. If this doesnt happen, refer to Add Bonus page.

Bonus List User Interface

The User Interface can be broken down into:

  1. Search: Search for Bonus added through Customer Username.
  2. USERNAME: Displays the Username of the NETTV Customer.
  3. MAC ADDRESS: Displays the Device/STB MAC Address of the respective username.
  4. OFFER NAME: Displays the name of the offer.
  5. INVOICE NUMBER: Displays the invoice number of the transaction when the customer paid their Internet bill.
  6. ADDED DATE: Displays the date and time of when the bonus was added.
  7. ADDED BY: Displays who added the bonus. If it's epayment.admin then it's an automated process which added the bonus automatically, if it's an exchange username of WorldLink staff then due to some issue, then respective staff added the bonus manually.

Invoices

Invoices section displays the logs of invoice information of the Customer.

Invoices User Interface

The User Interface can be broken down into:

  1. Customer ID: Displays the Username of the NETTV Customer.
  2. Customer Name: Displays the Name of the customer.
  3. Fiscal Year/Invoice no.: Displays in which fiscal year the invoice was generated along with the Invoice number.
  4. Payment Gateway: Displays the gateway of the payment.
  5. Product Name: Displays which product the customer purchased.
  6. Taxable Amount: Displays the taxable amount.
  7. Vat Amount: Displays the VAT amount.
  8. Total Amount: Displays the Total amount the customer paid.
  9. Quantity/Unit: ?????????????
  10. Transaction Date: Displays the date and time of the transaction.
  11. Created by: Displays the creator of the invoice.
  12. Branch: Displays the branch office name of the creator.

  13. Actions: Displays action buttons that performs certain function.
    In this case we only have a Print invoice button which will open the invoice in a ready to print (.pdf) format.

    Tip

    How to Print an invoice?

    1. Click the Print invoice at the end of the row of a desired Customer.
    2. A new tab with Invoice will open.
    3. Click the Print icon. (Depends upon the Internet Browser.)
    4. Click the Print button to begin printing the invoice.

  14. Download Button: Downloads the invoices records from a selected date range in an Excel format.

    Tip

    How to download invoices records?

    1. Select a starting date in From Date.
    2. Select an ending date in To Date.
    3. Click Download button to export the records in an Excel format.

  15. Refresh Button: Refreshes the Invoices UI.

  16. Filter Button: Filters the Invoices records based on given criterias.

    Here, users can filter the invoices records with

    1. Username: Customer's Username.
    2. Created by: Creator of the Invoice.
    3. Branch: Certain WorldLink branches.
    4. From Date: A starting date.
    5. To Date: An ending date.

If it's epayment.admin then it's an automated process which created the package bill automatically or if it's an exchange username of WorldLink staff then due to some issue, the respective staff added the bonus manually.   8. Status: Displays if the package is Active or not.

  9. ACTIONS: Displays action buttons that performs certain function.
    In this case we only have a cancel button which cancels the package billing.

    Tip

    How to cancel a package billing?

    1. Click CANCEL button of a desired Customer.
    2. Enter a valid cancel note in the Cancel note field.
    3. Click OK to confirm the Cancellation.

  10. Filter: Filters the Package Billing records based on given criterias.

Replace List

Replace List section displays the list of the Set Top Box (STB) replacement from one device to another.

Note

Replacement not only replaces a damaged STB with a new one but also migrates any NETTV Customer with Old system into a new one.

Replace List User Interface

The User Interface can be broken down into:

  1. Search: Search for Bonus added through Customer Username.
  2. USERNAME: Displays the Username of the NETTV Customer.
  3. OLD MAC ADDRESS: Displays the Device/STB MAC Address of the old Device/STB
  4. NEW MAC ADDRESS: Displays the Device/STB MAC Address of the new Device/STB.
  5. REPLACE DATE: Displays the date of the replacement made.
  6. REPLACE BY: Displays the name of the WorldLink Field Staff/Attendant (FA) who replaced the Device/STB.
  7. REASON: Displays the reason for which the replacement was made.
