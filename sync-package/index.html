Sync Package

Sync Package functionality is used to synchronize a customer's NETTV package with their corresponding WorldLink internet account.

How to Sync Package with the Internet

To Sync package with the Internet:

  1. Enter the Customer's username in the USERNAME field and select the correct username from the suggestions.
  2. Click the SYNC button.
  3. A confirmation prompt will appear. Click Continue to confirm the synchronization.
