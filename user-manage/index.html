User Management

Add Permission

Add Permission page is used to assign permissions which provies access to certain functionalities to the staff within the NETTV Dashboard.

Warning

Only give required permissions to the staff based on their role

How to Add permission

To add permission for the staff:

  1. In the Enter exchange username, enter the exchange username of the staff that needs permission.
  2. In the Enter permission name, enter the name of the permissions that needs to be assigned. You can find all the permissions that can be added below.
  3. Click ADD button.
  4. Click Continue to confirm. That's it!

Note

You can add multiple permissions to multiple staffs at once.
