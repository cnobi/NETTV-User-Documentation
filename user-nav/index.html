User

Create User

Create User functionality is used to re-map an existing username which due to some issues, didn't update in the Retail User's database.

How to create user?

To Create a User:

  1. Enter the Customer's username in the USERNAME field and select the respective username from the suggestions.
  2. Choose a respective MAC Address from Select MAC dropdown field.
  3. Click Create to create the user.

Warning

Even if a username already exists, users can still create a new user with that same username. This action will replace the existing user's package and its validity period. Before creating a new user, please make sure the username is not in the User List before proceeding.

Skip Uninstallation

Skip Uninstallation functionality is used when the Set Top Box (STB) cannot be uninstalled physically due to the device being lost, damaged or other unforeseen circumstances. This makes sure that the uninstallation process will continue without having to uninstall the STB Device.

Note

When a customer uninstalls their Internet then they are uninstalling the NETTV services simultaneously.

How to Skip Uninstallation?

To Skip Uninstallation for a customer:

  1. Enter the Customer's username in the USERNAME field and select the respective username from the suggestions.
  2. Choose a respective MAC Address from STB dropdown field.
  3. Enter remarks in the Remarks field.
  4. A prompt will appear asking for confirmation, click Continue to sync the package. Done!

Migrate

Migrate functionality is used to migrate old NETTV system users onto the new system.

Note

There are currently 2 NETTV Systems operating: Old and a New System. Majority of the customer base have been moved to the new system but some are still running on the old system.

To migrate a customer from Old to New NETTV System:

  1. Enter the Customer's username in the USERNAME field and select the respective username from the suggestions.
  2. Choose a respective MAC Address from STB dropdown field.
  3. Related information such as STB TYPE and MAPPING will be loaded after selecting the desired MAC Address.
  4. Click the Migrate button.
