    Users

    Users page is where all the Retail NETTV subscriber's record reside. This page also acts as the first point of troubleshooting where the details of all the customers can be viewed. NEED TO REVIEW and MAKE AMMENDMENTS

    Note

    The NETTV Dashboard exclusively maintains records for Retail Customers only and it's functionalities are specifically tailored to address them. It's important to note that Corporate NETTV Customers receive dedicated support from the Corporate department.

    Users UI

    Broken down in the screenshot above, the user interface contains a table with columns, buttons and a field through which Retail Users/Customer's information can be viewed and managed. NEED TO REVIEW and MAKE AMMENDMENTS

    1. ADD BONUS: Refer Add Bonus
    2. MIGRATE: Refer Migrate
    3. USERNAME: Displays the usernames that are subscribed to the NETTV service.
    4. MAC ADDRESS: Displays the Device/STB MAC Address of the respective username.
    5. CREATED DATE: Displays the date of the NETTV account created.
    6. ADDED BY: Displays the name of the WorldLink's staff who added the Customer's account.
    7. STATUS: Displays the Account Status of the respective username, whether the account is Active or Inactive.
    8. SEARCH: Users can search for customer's NETTV account information through their usernames or STB MAC Address using this search box.
    9. ACTION: Action Column contains three action buttons, each performing various tasks:

    Disable User

    Disables the respective customer from all the NETTV services. Make sure to provide a valid reason before disabling a customer. NEED TO REVIEW and MAKE AMMENDMENTS

    User Detail

    User detail brings up a window with two tabs containing various informations.

    Note

    There are currently two NETTV Systems operating: Old and a New System. Majority of the customer base have been moved to the new system but some are still running on the old system.

    The first tab, DETAIL(--- SYSTEM), displays the overall NETTV package information of the customer. NEED TO REVIEW and MAKE AMMENDMENTS

    1. PACKAGE NAME: Name of the NETTV package the customer is subscribed to.
    2. QUANTITY: Number of Days remaining in the customer's account.
    3. CREATED DATE
    4. EXPIRY DATE:
      • Channel: Expiry date of the NETTV. NEED TO REVIEW and MAKE AMMENDMENTS
      • App: Expiry date of the Mobile NETTV App. NEED TO REVIEW and MAKE AMMENDMENTS
    1. CREATED BY: Displays who created the account of the give NETTV Customer. It shows Staff's name if it was created by a WorldLink Staff, scripted if it was an automated creation or epayment.admin if it was created through an online payment procedure.
      NEED TO REVIEW and MAKE AMMENDMENTS
    2. STB: Shows if the device is either Primary or Secondary.

    The second tab, BILLING HISTORY displays every transaction history of the respective customer.

    Delete User

    This button deletes the customer account from the NETTV system.
